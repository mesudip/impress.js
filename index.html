64 bit Computer Architecture

Presenter

Sudip Bhattarai

64 bit Computing

  • Register size(int) - 64 bit
  • Data path width - 64 bit
  • Integer size - 64 bit
  • Code has 64 bit virtual memory addresses
  • Memory address width - 64 bit

Usage History

  • In supercomputers. eg (Cray-1) 1975
  • In RISC based workstations and servers. Eg. MIPS R4000, R8000, Power3 etc
  • In PCs introduced as x86-64 processors since 2003
  • The ARM architecture targeting smartphones eg iPhone 5S released on September 20, 2013

Major components of processor

  • General register unit
  • Control and decoding unit
  • Bus unit
  • Cache memory unit
  • Floating point register unit
  • Issue ports

New Instructions

  • Load Reserved - LRC, LRVR
  • Multiply Logical - ML, MLR
  • Divide Logical - DL, DLR
  • Add Logical w/Carry - ALC
  • Subtract Logical w/Borrow - SLB
  • Store Reserved - STRV
  • Rotate Left Single Logical- RRL

Advantage

  • 64-bit processors have 64 bit data lines
  • Speed and execution is both faster in 64-bit processors
  • 64 bit cpu executing in 32 bit mode is faster
